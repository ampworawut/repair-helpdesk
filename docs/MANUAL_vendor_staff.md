## 1. การเข้าสู่ระบบ

1. เปิดเว็บเบราว์เซอร์ ไปที่ URL ของระบบ
2. กรอก **อีเมล** และ **รหัสผ่าน**
3. กดปุ่ม **"เข้าสู่ระบบ"**

> 💡 หากลืมรหัสผ่าน ให้ติดต่อ Admin

---

## 2. หน้าแดชบอร์ด

เมื่อเข้าสู่ระบบ จะเห็นภาพรวม:

- **การ์ดสรุป:** จำนวนเคสที่เปิดอยู่ · คำเตือน SLA · SLA ที่เกินกำหนด · เคสที่พักไว้
- **กราฟแท่ง:** แนวโน้มเคสรายสัปดาห์
- **กราฟวงกลม:** สัดส่วนเคสตามหมวดหมู่
- **ตาราง:** 10 เคสล่าสุด

### เมนูด้านข้าง (Vendor Staff)

| เมนู | คำอธิบาย |
|------|----------|
| 🏠 แดชบอร์ด | ภาพรวมระบบ |
| 📋 เคสทั้งหมด | รายการเคสของกลุ่มคุณ |
| 👤 โปรไฟล์ | ข้อมูลส่วนตัว |

---

## 3. รายการเคส

คลิก **"📋 เคสทั้งหมด"** เพื่อดูรายการเคสที่เกี่ยวข้องกับกลุ่มผู้ให้เช่าของคุณ

### การกรองและค้นหา
- ค้นหาด้วยเลขเคส / หัวข้อ / รหัสเครื่อง
- กรองตามสถานะ, ความสำคัญ, ช่วงวันที่

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| เลขเคส | REP-ปี-ลำดับ (เช่น REP-26-0051) |
| หัวข้อ | ชื่อปัญหา |
| หมวดหมู่ | ประเภทปัญหา |
| เครื่อง | รหัสเครื่อง |
| ผู้ให้เช่า | บริษัทผู้ให้เช่า |
| ผู้แจ้ง | ชื่อผู้สร้างเคส |
| สถานะ | สถานะปัจจุบัน |
| ความสำคัญ | ปกติ / ด่วน |
| วันที่ | วันที่สร้าง |

## 4. หน้ารายละเอียดเคส

คลิกเคสใดๆ เพื่อเปิดหน้ารายละเอียด

### ข้อมูลด้านซ้าย
- เลขเคส, หัวข้อ, สถานะ, ความสำคัญ, หมวดหมู่
- เครื่อง, ผู้ให้เช่า, สถานที่, ผู้แจ้ง, วันที่
- ช่างที่ได้รับมอบหมาย
- **⏱️ SLA** — เวลาตอบรับและเวลาช่างมาถึง (🟢 ปกติ / 🟡 ใกล้หมด / 🔴 เกินกำหนด)

### การเปลี่ยนสถานะ

นี่คือหน้าที่หลักของคุณ — อัปเดตสถานะเคสตามความคืบหน้า:

| สถานะปัจจุบัน | เปลี่ยนเป็น | เมื่อไหร่ที่ควรใช้ |
|---------------|------------|-------------------|
| รอดำเนินการ | 📌 **ตอบรับเรื่อง** | เมื่อรับทราบเคสแล้ว |
| ตอบรับเรื่อง | 🔧 **เริ่มดำเนินการ** | เมื่อช่างไปถึงหน้างาน |
| กำลังดำเนินการ | ✅ **เสร็จสิ้น** | เมื่อซ่อมเสร็จแล้ว |
| กำลังดำเนินการ | ⏸️ **พัก** | เมื่อต้องรออะไหล่หรือปัจจัยอื่น |
| พักการดำเนินการ | 🔧 **กลับมาดำเนินการ** | เมื่อกลับมาทำงานต่อ |

> 💡 ทุกครั้งที่เปลี่ยนสถานะ ระบบจะบันทึกประวัติและส่งการแจ้งเตือนไปยัง LINE โดยอัตโนมัติ

### วงจรสถานะเคส

```
รอดำเนินการ → ตอบรับเรื่อง → เริ่มดำเนินการ → เสร็จสิ้น → ปิดรายการ
                                  ↓
                               พักชั่วคราว
```

### ประวัติการดำเนินการ (Activity Log)
ไทม์ไลน์แสดงทุกการกระทำที่เกิดขึ้นกับเคส

### ความคิดเห็น (Comments)
- **ทั่วไป** — ทุกคนเห็น
- **ภายใน** — เฉพาะ Admin, Supervisor, Vendor Staff (ใช้สำหรับบันทึกข้อมูลทางเทคนิค)

### การแนบรูปภาพ
คลิก **"แนบรูป"** → เลือกไฟล์ → รูปถูกบีบอัดอัตโนมัติ → กดส่ง
ใช้สำหรับแนบรูปหน้างาน เช่น สภาพเครื่องก่อน/หลังซ่อม

## 5. การใช้งานผ่าน LINE

คุณสามารถทำงานผ่าน LINE Group ได้โดยไม่ต้องเปิดเว็บ:

### ตรวจสอบสถานะเคส
พิมพ์ **เลขเคส** ใน LINE Group (เช่น `0051` หรือ `REP-26-0051`) ระบบจะตอบกลับด้วยสถานะล่าสุด

### ส่งรูปภาพ
ส่งรูปภาพใน LINE Group เพื่อแนบเข้ากับเคสล่าสุดของกลุ่มคุณโดยอัตโนมัติ

### รับการแจ้งเตือน
ระบบจะส่งข้อความแจ้งเตือนเมื่อ:
- มีเคสใหม่เข้ามา
- มีการมอบหมายช่าง
- SLA ใกล้หมดหรือเกินกำหนด
- มีการขอยืนยันผลการแก้ไข

## 6. การจัดการโปรไฟล์

คลิกชื่อผู้ใช้มุมล่างซ้าย → **"โปรไฟล์"**

- เปลี่ยนรหัสผ่าน (ต้องกรอกรหัสผ่านปัจจุบัน)
- ออกจากระบบ
