## 1. การเข้าสู่ระบบ

1. เปิดเว็บเบราว์เซอร์ ไปที่ URL ของระบบ
2. กรอก **อีเมล** และ **รหัสผ่าน**
3. กดปุ่ม **"เข้าสู่ระบบ"**

> 💡 หากลืมรหัสผ่าน ให้ Admin เป็นผู้ตั้งรหัสผ่านใหม่ให้

---

## 2. หน้าแดชบอร์ด

เมื่อเข้าสู่ระบบ จะเห็นภาพรวมของระบบ:

- **การ์ดสรุป:** จำนวนเคสที่เปิดอยู่ · คำเตือน SLA · SLA ที่เกินกำหนด · เคสที่พักไว้
- **กราฟแท่ง:** แนวโน้มเคสรายสัปดาห์
- **กราฟวงกลม:** สัดส่วนเคสตามหมวดหมู่
- **ตาราง:** 10 เคสล่าสุด

### เมนูด้านข้าง (Admin เห็นทั้งหมด)

| เมนู | คำอธิบาย |
|------|----------|
| 🏠 แดชบอร์ด | ภาพรวมระบบ |
| 📋 เคสทั้งหมด | รายการเคสทั้งหมด |
| ➕ แจ้งซ่อมใหม่ | สร้างเคสใหม่ |
| 📦 สร้างเคสแบบกลุ่ม | Batch สร้างเคสหลายรายการ |
| ⚙️ จัดการระบบ | เมนูผู้ดูแลระบบ |
| 📊 รายงาน | กราฟและสถิติ |
| 👤 โปรไฟล์ | ข้อมูลส่วนตัว |

---

## 3. การแจ้งซ่อม (สร้างเคส)

1. คลิก **"➕ แจ้งซ่อมใหม่"**
2. **ค้นหาเครื่อง:** พิมพ์รหัสเครื่องหรือชื่อเครื่อง
3. **กรอกข้อมูล:**
   - **หัวข้อ** — อธิบายปัญหา (เช่น "จอฟ้า", "เปิดไม่ติด")
   - **สถานที่รับบริการ** — เลือกจากรายการ
   - **ความสำคัญ** — ปกติ หรือ ด่วน
   - **รายละเอียด** — เพิ่มเติม (ไม่บังคับ)
4. **แนบรูปภาพ** (ถ้ามี) — สูงสุด 10 รูป, ไม่เกิน 5MB/รูป
5. กด **"สร้างเคส"**

> 💡 ระบบจัดหมวดหมู่ให้อัตโนมัติจากหัวข้อและรายละเอียด
> ⚠️ ไม่สามารถสร้างเคสซ้ำสำหรับเครื่องที่มีเคสค้างอยู่

---

## 4. การสร้างเคสแบบกลุ่ม (Batch)

สำหรับสร้างเคสหลายรายการพร้อมกัน มี 4 ขั้นตอน:

1. **เลือกเครื่อง** — ค้นหาและเลือกหลายเครื่อง
2. **ตั้งรายละเอียด** — กำหนดหัวข้อ รายละเอียด ความสำคัญให้แต่ละเคส
3. **ตรวจสอบ** — ดูตัวอย่างก่อนสร้าง
4. **สร้าง** — กดยืนยัน ระบบสร้างทั้งหมดและแสดงผลลัพธ์

---

## 5. รายการเคสทั้งหมด

คลิก **"📋 เคสทั้งหมด"** เพื่อดูรายการ

### การกรองและค้นหา
- ค้นหาด้วยเลขเคส / หัวข้อ / รหัสเครื่อง
- กรองตามสถานะ, ความสำคัญ, ช่วงวันที่

### คอลัมน์ในตาราง

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| เลขเคส | REP-ปี-ลำดับ (เช่น REP-26-0051) |
| หัวข้อ | ชื่อปัญหา |
| หมวดหมู่ | ประเภทปัญหา |
| เครื่อง | รหัสเครื่อง |
| ผู้ให้เช่า | บริษัทผู้ให้เช่า |
| ผู้แจ้ง | ชื่อผู้สร้างเคส |
| สถานะ | สถานะปัจจุบัน |
| ความสำคัญ | ปกติ / ด่วน |
| วันที่ | วันที่สร้าง |

---

## 6. หน้ารายละเอียดเคส

คลิกเคสใดๆ เพื่อเปิดหน้ารายละเอียด

### ข้อมูลด้านซ้าย
- เลขเคส, หัวข้อ, สถานะ, ความสำคัญ, หมวดหมู่
- เครื่อง, ผู้ให้เช่า, สถานที่, ผู้แจ้ง, วันที่
- ช่างที่ได้รับมอบหมาย
- **⏱️ SLA** — เวลาตอบรับและเวลาช่างมาถึง (🟢 ปกติ / 🟡 ใกล้หมด / 🔴 เกินกำหนด)

### การเปลี่ยนสถานะ

| สถานะปัจจุบัน | เปลี่ยนเป็น |
|---------------|------------|
| รอดำเนินการ | 📌 ตอบรับเรื่อง · ❌ ยกเลิก |
| ตอบรับเรื่อง | 🔧 เริ่มดำเนินการ · ⏸️ พัก · ❌ ยกเลิก |
| กำลังดำเนินการ | ✅ เสร็จสิ้น · ⏸️ พัก · ❌ ยกเลิก |
| พักการดำเนินการ | 🔧 กลับมาดำเนินการ · ❌ ยกเลิก |
| เสร็จสิ้น | 🔒 ปิดรายการ |

### การมอบหมายช่าง
กดปุ่ม **"มอบหมายช่าง"** → เลือกช่างจากรายชื่อ (กรองตามกลุ่มผู้ให้เช่า)

### ประวัติการดำเนินการ (Activity Log)
ไทม์ไลน์แสดงทุกการกระทำ: สร้างเคส, เปลี่ยนสถานะ, มอบหมายช่าง, เปลี่ยนหมวดหมู่, ยืนยันผล, ความคิดเห็น, รูปภาพ

### ความคิดเห็น (Comments)
- **ทั่วไป** — ทุกคนเห็น
- **ภายใน** — เฉพาะ Admin, Supervisor, Vendor Staff

### การแนบรูปภาพ
คลิก **"แนบรูป"** → เลือกไฟล์ → รูปถูกบีบอัดอัตโนมัติ → กดส่ง

### การยืนยันผลการแก้ไข
เมื่อเคส "เสร็จสิ้น" ผู้แจ้งสามารถ:
- **✅ ยืนยัน** — พร้อมปิด
- **❌ ไม่ยอมรับ** — กลับไป "กำลังดำเนินการ"

## 7. รายงาน (Reports)

เมนู **"📊 รายงาน"** แสดง:

- การ์ดสรุป (เคสทั้งหมด, เปิดอยู่, ปิดแล้ว, เกิน SLA, ข้อความ LINE)
- กราฟแนวโน้มข้อความ LINE รายเดือน
- สัดส่วนเคสตามหมวดหมู่
- สรุปเคสรายเดือน (เปิด vs ปิด)
- สัดส่วน SLA (ผ่าน/เกิน/รอ)
- MTTR (เวลาเฉลี่ยในการซ่อม)
- ประสิทธิภาพผู้ให้เช่า
- ตารางสรุปตามสถานะ
- ปุ่ม **ส่งออก CSV**

## 8. การจัดการระบบ

### 8.1 จัดการผู้ใช้
- สร้างผู้ใช้ใหม่ (อีเมล, รหัสผ่าน, ชื่อ, บทบาท, ผู้ให้เช่า)
- แก้ไขข้อมูล / เปลี่ยนรหัสผ่าน / เปิด-ปิดการใช้งาน / ลบผู้ใช้

### 8.2 จัดการผู้ให้เช่า (Vendors)
- เพิ่ม/แก้ไขบริษัท (ชื่อ, รหัส, ผู้ติดต่อ, อีเมล, โทรศัพท์)
- กำหนดกลุ่มและประเภท (บริษัท/บริษัทย่อย/นามแฝง)

### 8.3 จัดการกลุ่มผู้ให้เช่า (Vendor Groups)
- สร้างกลุ่ม เพิ่มผู้ให้เช่าเข้ากลุ่ม
- ตั้งค่าการแจ้งเตือน LINE แยกตามกลุ่ม (เปิด/ปิดแต่ละประเภท)
- ตรวจหา LINE Group ID

### 8.4 จัดการเครื่องคอมพิวเตอร์ (Assets)
- เพิ่ม/แก้ไขเครื่อง (รหัส, ซีเรียล, รุ่น, ผู้ให้เช่า, สถานที่, ผู้ใช้, สถานะ)
- **นำเข้าจาก Excel** (XLSX)

### 8.5 จัดการสถานที่ให้บริการ
- เพิ่ม/แก้ไข/ลบสถานที่ (ชื่อ, อาคาร, ชั้น) จัดเรียงลำดับได้

### 8.6 จัดการเทมเพลต
- สร้างเทมเพลตสำหรับกรอกเคสล่วงหน้า (หัวข้อ, รายละเอียด, ความสำคัญ, หมวดหมู่)

### 8.7 ตั้งค่า SLA
- ชั่วโมงตอบรับ (ค่าเริ่มต้น: 4 ชม.)
- ชั่วโมงช่างมาถึง (ค่าเริ่มต้น: 18 ชม.)
- เวลาทำงาน (เริ่ม-สิ้นสุด, ค่าเริ่มต้น: 8:30-17:30)

### 8.8 จัดการวันหยุด
- เพิ่ม/แก้ไข/ลบวันหยุด (วันที่, ชื่อ)
- นำเข้าจาก CSV
- วันหยุดถูกยกเว้นจากการคำนวณ SLA

### 8.9 ส่งออกข้อมูล
- ส่งออกฐานข้อมูลเป็น SQL หรือ JSON
- เรียกดู/ดาวน์โหลดไฟล์แนบจาก S3 Storage

### 8.10 ตั้งค่าอีเมล
- เปิด/ปิดการส่งอีเมล
- ตั้งค่า Provider (Resend หรือ SMTP)
- เลือกเหตุการณ์ที่ต้องการให้ส่งอีเมล

## 9. การใช้งานผ่าน LINE

- **รับการแจ้งเตือน:** เมื่อมีเหตุการณ์ (สร้างเคส, เปลี่ยนสถานะ, SLA เตือน) ระบบส่งข้อความไปยัง LINE Group
- **ตรวจสอบสถานะ:** พิมพ์เลขเคสใน LINE Group (เช่น `0051`) เพื่อดูสถานะ
- **ส่งรูปภาพ:** ส่งรูปใน LINE Group เพื่อแนบเข้ากับเคสล่าสุด

> 💡 เปิด/ปิดการแจ้งเตือนแต่ละประเภทได้ที่หน้า "จัดการกลุ่มผู้ให้เช่า"

## 10. การจัดการโปรไฟล์
